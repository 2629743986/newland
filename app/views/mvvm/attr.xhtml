attr绑定

attr绑定的参数总是一个对象，里面是它要添加的属性名与属性值。

示例 青蛙公主

HTML：

    <a bind="attr: { href: url, title: details }">
        青蛙公主
    </a>

JS:

    $.require("ready,./more/avalon", function(){
        var model = $.ViewModel({
            url: "http://yyets.com/showresource-juji-1300.html",
            details: "全11回/2012春季日剧/YYeTs人人影视字幕组/"+
                "中日双语字幕[HR-HDTV,RMVB,720P]"
        })
        $.View(model);
    })
