模块加载模块

描述：

mass Framework的最核心的种子模块，提供构筑框架的最基本工具， 包括模块加载系统，mix机制， 数组化，类型判定，别名机制，调试机制，domReady......

主要API介绍

exports rword mix
API 解说
html 返回HTML节点
head 返回HEAD节点
mix 合并一个或多个对象到第一个参数（它也必须是对象）中
rword 一个有用的正则，将字符串以空白或“，”切割成一个个单词
mass 用于储存框架当前的版本号
exports 多库共存
